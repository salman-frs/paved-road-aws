# orders-api

`orders-api` is the reference service generated by the paved-road platform template. It exists to prove the full scaffold, build, infra, GitOps, observability, and scorecard path.

## Included Defaults

- Fastify HTTP API with health endpoints
- structured JSON logging
- Prometheus metrics
- OpenTelemetry tracing
- Helm chart and ArgoCD application
- service-local Terraform wrapper
- baseline dashboard, alerts, and runbook

## Local Development

```bash
pnpm install
pnpm --filter @paved-road/orders-api dev
pnpm --filter @paved-road/orders-api test
pnpm --filter @paved-road/orders-api build
```
