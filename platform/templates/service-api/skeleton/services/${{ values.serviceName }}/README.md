# ${{ values.serviceName }}

`${{ values.serviceName }}` is a production-ready TypeScript HTTP API scaffold generated by the paved-road platform template.

## What It Includes

- Fastify HTTP API
- structured JSON logging
- Prometheus metrics
- OpenTelemetry tracing
- Helm chart and ArgoCD deployment path
- service-local Terraform wrapper
- dashboard, alerts, runbook, and Backstage metadata

## Local Development

```bash
pnpm install
pnpm --filter @paved-road/${{ values.serviceName }} dev
pnpm --filter @paved-road/${{ values.serviceName }} test
pnpm --filter @paved-road/${{ values.serviceName }} build
```
